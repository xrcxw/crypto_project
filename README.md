crypto\_project - cервис для начинающих инвесторов, который анализирует риск криптоактивов на основе исторической волатильности.





Рабочий проект: https://xrcxw.pythonanywhere.com/



\## О проекте



Автоматически получает данные о топ-10 криптовалютах с CoinGecko API, рассчитывает годовую волатильность и предоставляет наглядную визуализацию для принятия инвестиционных решений.



\## Возможности:

\- Анализ волатильности 10 криптовалют

\- Интерактивные графики динамики цены

\- Избранное для отслеживания монет

\- Текстовые рекомендации по уровню риска

\- Регистрация и авторизация пользователей



\## Стек технологий



| Категория        | Технологии                   |

|------------------|------------------------------|

| \*\*Backend\*\*      | Python 3.10, Django 4.2      |

| \*\*База данных\*\*  | SQLite                       |

| \*\*Аналитика\*\*    | NumPy (расчёт волатильности) |

| \*\*Визуализация\*\* | Plotly                       |

| \*\*API\*\*          | CoinGecko                    |

| \*\*Frontend\*\*     | Bootstrap 5.3                |





\## Локальный запуск



\# Клонирование

git clone https://github.com/xrcxw/crypto\_project.git

cd crypto\_project



\# Виртуальное окружение

python -m venv venv

source venv/bin/activate  # или venv\\Scripts\\activate



\# Зависимости

pip install -r requirements.txt



\# Миграции

python manage.py migrate



\# Создание администратора

python manage.py createsuperuser



\# Запуск

python manage.py runserver
