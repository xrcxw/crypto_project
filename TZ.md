## 1. Цель проекта

Разработать веб-приложение для анализа волатильности криптовалют, которое получает данные из внешнего API, рассчитывает статистические метрики риска и предоставляет визуальную аналитику.


## 2. Целевая аудитория

Начинающие инвесторы в криптовалюты, трейдеры, пользователи, интересующиеся рынком цифровых активов.


## 3. Функциональные требования


| Функция              | Описание                                    |

|----------------------|---------------------------------------------|

| Загрузка из API      | Получение топ-10 криптовалют с CoinGecko    |

| История цен          | Хранение цен за 7 дней                      |

| Расчёт волатильности | Годовая волатильность (σ \* √365)           |

| Рекомендации         | Классификация риска: высокий/средний/низкий |

| Визуализация         | Интерактивный график                        |

| Избранное            | Watchlist для авторизованных пользователей  |


## 4. Структура БД

|------|------------|--------------|

## 5. Алгоритмы

Волатильность:

1. Логарифмические доходности: ln(P\_t / P\_{t-1})

2. Стандартное отклонение доходностей

3. Умножение на √365

4. Перевод в проценты


Классификация:

- >80% - высокий риск

- 40-80% - средний риск

- <40% - низкий риск
